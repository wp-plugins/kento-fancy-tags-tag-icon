Tag icon and tag description on hover box.

== Description ==

Add tag icon and tag desription to your post and tag page will boost SEO efforts and beautiful tag list.

Live Preview: http://kentothemes.com/demo/kento-fancy-tags/





Plugin Features

* Tag icon for each tag.
* Tag icon on post.
* Tag description popup box with tag icon.
* post count for tag on hover box. 



== Installation ==


1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. Before test plugin you need to add some tag on your single post.
4. After acivate plugin you will see menu at left menu bar "KFT Settings" please set value for<br /> 

<strong>Popup box top position on hover:</strong> an integer top from mouse position(could be use nagetive)<br /> 
<strong>Popup box left position on hover:</strong>an integer left from mouse position(could be use nagetive)<br /> 
<strong>Display popup box:</strong> to hide hover box on mouse hover on tag link<br /><br /> 

<strong>How to display tag icon ?</strong><br /> 
To display tag icon you must save an .png image exactly same as image name of tag slug to <strong>"tag-images"</strong> folder or directory on this plugin.

for example if you want to display tag icon for "facebook" on your post then you have to save an image  "facebook.png" to <strong>"tag-images"</strong>(inside this plugin, folder alredy exist) folder or directory then it will automatically added before tag name on single post tag list.

<strong>How to display tag description ?</strong><br /> 
To display tag description you need to fill tag desription under <strong>Posts > Tags > Edit > Description</strong>
